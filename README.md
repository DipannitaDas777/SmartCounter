# Smart Counter App

##  Overview

Smart Counter is an Android application developed using Kotlin and Jetpack Compose. It allows users to increment, decrement, and reset a counter with a simple and clean user interface.

##  Features

- Increment counter value
- Decrement counter value
- Reset counter to zero
- Simple and user-friendly UI
- Input validation (prevents invalid states)
- Real-time UI updates using Jetpack Compose

## ️ Technologies Used

- Kotlin
- Jetpack Compose
- Android Studio
- Material Design 3

##  How to Run

1. Clone this repository.
2. Open the project in Android Studio.
3. Allow Gradle to sync.
4. Run the application on an emulator or Android device.
